Hh_df {VESimHouseholds}R Documentation

Household data from Census PUMS

Description

A household dataset containing the data used for estimating the CreateHouseholds, PredictWorkers, PredictLifeCycle, PredictIncome, and PredictHouseType modules derived from from year 2000 PUMS data for Oregon.

Usage

Hh_df

Format

A data frame with 65988 rows and 17 variables (there may be a different number of rows if PUMS datasets are used for different areas):

Age0to14

number of persons in 0 to 14 age group

Age15to19

number of persons in 15 to 19 age group

Age20to29

number of persons in 20 to 29 age group

Age30to54

number of persons in 30 to 54 age group

Age55to64

number of persons in 55 to 64 age group

Age65Plus

number of persons 65 years or older

Wkr15to19

number of workers in 15 to 19 age group

Wkr20to29

number of workers in 20 to 29 age group

Wkr30to54

number of workers in 30 to 54 age group

Wkr55to64

number of workers in 55 to 64 age group

Wkr65Plus

number of workers 65 years or older

AvePerCapInc

average per capita income of PUMA, nominal $

HouseType

housing type (SF = single family, MF = multifamily)

Income

annual household income, nominal 1999$

HhSize

number of persons in household

HhType

household type (Reg = regular household, Grp = group quarters)

HhWeight

household sample weight

Source

CreateEstimationDatasets.R script.
